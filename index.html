Bootstrap custom grid examples

Basic grid layouts to get you familiar with building within the Bootstrap grid system.

Code example

HTML markup

<div class="wrapper">
	<div class="content-main">custom-grid - 2 columns in grid of 10 columns</div>
	<div class="content-main">custom-grid - 2 columns in grid of 10 columns</div>
	<div class="content-main">custom-grid - 2 columns in grid of 10 columns</div>
	<div class="content-main">custom-grid - 2 columns in grid of 10 columns</div>
	<div class="content-main">custom-grid - 2 columns in grid of 10 columns</div>
</div>

less

.wrapper {
  .make-row-custom();
}
.content-main {
	//2 columns on 10 columns grid for small devices
	.make-sm-column-custom(2,10);
	
	//2 columns on 10 columns grid for medium devices
	.make-md-column-custom(2,10);
	
	//2 columns on 10 columns grid for large devices
	.make-lg-column-custom(2,10);
}

*Is the similar method used by Bootstrap. make-lg-column(), make-md-column(), but you can pass grid value
